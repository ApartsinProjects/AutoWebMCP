## Error Handling

Every tool call verifies the browser is on a Google Forms page before executing. If not, returns a `state_error` with the current URL.

All errors return structured responses with:
- `success`: boolean
- `error`: error message
- `category`: one of `selector_not_found`, `timeout`, `state_error`, `unknown`
- `hint`: optional recovery suggestion

Mutation tools (`set_form_title`, `set_form_description`, `set_document_title`) include readback verification.
